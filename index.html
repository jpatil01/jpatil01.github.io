Projects

World Generation

A program that can generate a random map of 2D terrain. There are various versions ranging from the incredibly dubious to impressively realistic as I used different algorithms.

This one uses a cellular automaton. This one uses Perlin noise. And this one uses midpoint displacement and has a 3d representation with three.js.

Universal Gravitation Simulator

A program that simulates orbit and gravitational force between planets in space.

You can check it out here.

Raycaster

This is a raycaster, a program that generates "3D" graphics using rays that hit walls.

Crocodile Coding

I am co-founder and executive director of a tutoring organization called Crocodile Coding. At Crocodile Coding, we taught Python and Scratch to elementary and middle schoolers from a homemade curriculum.
